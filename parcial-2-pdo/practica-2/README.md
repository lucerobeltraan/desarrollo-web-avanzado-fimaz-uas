Práctica 2 - Parcial 2
Objetivo
Se implemento el uso de transacciones PHP-PDO.
Demostrar el uso de transacciones COMMIT y ROLLBACK utilizando la extensión PDO en PHP para garantizar la integridad de los datos al realizar múltiples operaciones dependientes en una base de datos MySQL.


Tecnologías utilizadas
PHP 8+
MySQL

Instrucciones de ejecución
Pasos para ejecutar el proyecto localmente.
Paso 1: Se prepararon los archivos en la ruta correspondiente.
Paso 2: Se crearon las tablas en MYSQL.
Paso 3: Se creo la conexion de la base de datos y se le dio el formato en el index.
Paso 4: Se ejecuta y se realizan pruebas para verificar su funcionamiento.
