Práctica 2 - Parcial 1
Objetivo
En la práctica 2 se crearon dos clases: usuario como clase base y admin como clase hija. La clase admin hereda de usuario. 
Se crearon los archivos admin.php, usuario.php, index.php y README.md dentro de la práctica.

Tecnologías utilizadas
PHP 8+

Instrucciones de ejecución
Instrucciones de ejecución 
Paso 1.- Tener instalado previamente XAMPP con el servicio Apache activo. 
Paso 2.- descargar el parcial y abrir el index.php en el localhost.
