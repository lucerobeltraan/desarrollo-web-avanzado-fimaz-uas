# Práctica 4 - Mini Sistema POO en PHP

Objetivo
Construir un sistema orientado a objetos en PHP que simule un examen parcial, aplicando:
- Encapsulamiento
- Herencia
- Polimorfismo
- Validación de datos
- Manejo de excepciones
- Salida en HTML

Requisitos
- PHP 8+
- Servidor local XAMPP

Ejecución
Ruta: `http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-4/index.php`

Evidencia esperada
- Tabla HTML con usuarios válidos.
- Mensaje controlado de error por correo inválido.
