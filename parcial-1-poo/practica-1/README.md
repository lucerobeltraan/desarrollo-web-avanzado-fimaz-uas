Práctica 1 - Parcial 1
Objetivo
Cree un archivo practica-1 y en este archivo contiene el código que se ejecutará en el navegador.

Dentro del archivo se creó una clase, que funciona para organizar los datos y funciones, en el 
cual se declararon atributos privados para proteger que los datos no se modifiquen directamente y
para acceder a ellos se crearon los métodos **set** (para asignar valores) y **get** (para obtener valores)
con el fin de un manejo seguro y correcto de la información.

Después de definir la clase, se creó un objeto usando la palabra new. Con ese objeto se asigna los 
valores usando los métodos set y se muestran los datos con los métodos get. En el navegador se puede
ingresar como http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-1/, el resultado
que aparece en la pantalla muestra los datos del objeto, lo que confirma que el programa funciona correctamente.


Tecnologías utilizadas
PHP 8+

Instrucciones de ejecución
Paso 1.- Tener instalado previamente XAMPP con el servicio Apache activo.
Paso 2.- descargar el parcial y abrir el index.php en el localhost.
